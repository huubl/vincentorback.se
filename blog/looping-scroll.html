How to make a looping scroll

August 19, 2015

I had an idea for a client project that never made it into production, but I just had to build a small JavaScript proof of concept anyways :)

These looping scroll websites seems to be popping up more often now I get the feeling that we might just be in for a new trend in the measures of parallax trend we’ve seen and grown tired of.

I made a small demo you can check out below. Get scrolling!

See the Pen Scroll loop by Vincent Orback (@vincentorback) on CodePen.

The trick is to have a set of cloned elements on the bottom of the path which will make the jump to the top look seamless when you reach the bottom.

Test the demo in fullscreen here and of course, browse the code here.

If you’ve seen any more websites using this technique, please leave a link in the comments! Also, report a bug if you’ve seen one :)
