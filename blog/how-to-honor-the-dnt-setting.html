How to honor the DNT setting

December 1, 2014

DNT, or Do Not Track is a header sent by the browser which should be able to tell any software to not track any information sent by that request.

It’s available as a setting in most browsers but still hasn’t been accepted as a standard by the W3C.

If you want people to be able to decide what they do online should or shouldn’t be monitored, use this code-snippet to see if a DNT-setting is turned on in your code.

var DoNotTrack = navigator.doNotTrack === 'yes'
  || navigator.doNotTrack === '1'
  || window.doNotTrack === '1'
  || navigator.msDoNotTrack === '1'; // IE9 and IE10

if (!DoNotTrack) {
// Do your tracking here...
}

Services like Google Analytics doesn’t honor the DNT fully so be nice and check with your user before tracking them!
